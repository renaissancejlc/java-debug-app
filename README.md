# Java Debugging Practice Tool

🔗 Live App: https://debugjava.netlify.app/

## Overview

This project is a lightweight, interactive web application designed to help developers practice **real-world Java debugging skills**.

Instead of solving algorithm problems, users are presented with **buggy code snippets** and must identify what is wrong — just like in real interviews and production scenarios.

---

## Purpose

Most coding prep focuses heavily on algorithms, but modern engineering interviews (and real jobs) often test:

- Debugging unfamiliar code
- Understanding broken systems
- Identifying subtle bugs quickly
- Explaining reasoning clearly

This tool was built to train those exact skills.

---

## Core Features

- 🧠 **Bug Identification Practice**  
  View Java code snippets and determine what the bug is

- 🎯 **Searchable Answer Selection**  
  Type to filter and select the bug you believe is present

- ⏱️ **Timed Questions**  
  Practice under time pressure similar to interviews

- 🔍 **Reveal + Explanation Mode**  
  Reveal the correct bug and expand to see detailed explanations

- 🎲 **Random Play Mode**  
  Jump into a random problem instantly

- 🧩 **Category Filtering**  
  Practice by topic (Spring, Concurrency, APIs, etc.)

- 📚 **Wide Problem Coverage**  
  Includes beginner → senior-level issues:
  - NullPointerExceptions in different forms
  - Concurrency bugs
  - Transaction issues
  - Distributed systems edge cases
  - Caching and performance problems
  - API design pitfalls

---

## Tech Stack

- **Frontend:** React (Vite)
- **Styling:** Tailwind CSS
- **State Management:** React hooks
- **Data Source:** Static JSON problem bank
- **Hosting:** Netlify

---

## How It Works

1. Load a problem from the problem bank
2. Read the code and symptom
3. Identify the bug
4. Select your answer from the dropdown
5. Reveal the correct answer
6. Expand explanation to understand the reasoning
